# Circle

## Steps to run the project

- Clone the repository.
- Install nodemon `npm install -g nodemon`
- Run `npm install` for downloading the dependencies
- Run `nodemon server.js`
- The app is running on `localhost:5000`

## Tech stack used

- NodeJS
- PeerJS
- Socket.io
- Bootstrap
